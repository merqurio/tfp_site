# [iomed.es](http://iomed.es)

Install process:
`npm install & bower install`

Local serve:
`gulp serve`

*HTML development should be done in `app/templates`. It's automatically rendered while serving or in the build process.*

Build for serving:
`gulp build`

Deploy to GH:
`gulp deploy`
